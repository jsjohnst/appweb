Native APIs

The Ejscript Native APIs consist of C language headers, function prototypes, structure definitions and defines. They are divided into one of these groups.

Area Description
Appweb Appweb Http API
Ejscript Interpreter Ejscript Language (Javascript) Interpreter API
Portable Runtime Foundation Platform Runtime API
