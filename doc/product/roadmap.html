Appweb Roadmap

The Appweb roadmap is created by listening to user input. As such, it is subject to change. This roadmap is not a commitment to product, but the current thinking about future directions.

Those who shout loudest -- often get what they ask for. To request a feature, please send email to:

dev@mbedthis.com

Current Roadmap Tasks

This list is not in any particular order.

  • Backporting the 3.0 release to Uclinux
  • Administration Management Console
  • MySQL database connector
  • Model connector for JavaScript objects
  • More Ajax library connectors
  • Python module
  • Ruby module
  • Compressed ROM pages
  • SNMP manageability

What's in the Next Release

The next release in the stable branch will be the Embedthis Appweb 2.5. Patches to the stable branch are released only as issues require it.

For the development branch, we will continue with incremental alpha and beta releases on the road toward a production 3.0 release which is expected in Q2 2009.

Please tell us what you would like to see at dev@mbedthis.com.
