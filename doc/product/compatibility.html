Compatibility

Appweb supports the following standards:

  • HTTP/1.1
  • CGI 1.1
  • Form-based File Upload in HTML
  • ECMA-262 for server side JavaScript

API Compatibility

The Appweb APIs are evolving and are thus subject to change between minor releases. The documentation provides a Stability Classification for each API group that reflects the stability with regard to backwards compatibility.

We do not provide a blanket commitment to backward compatibility. Rather, we adopt a more surgical approach on an API by API basis.

Appweb 2

Appweb 2 is a stable release that in now in maintenance mode. The core programming APIs are relatively stable and are not expected to change significantly. Appweb 2 will be maintained until the end of 2009.

Appweb 3

Appweb 3 is a major change relative to Appweb 2. Most of the APIs have changed in some capacity and you should regard Appweb 3.X as a next generation web server. You can migrate code from Appweb 2 to Appweb 3, however you should expect to have significant refactoring work when transitioning.
