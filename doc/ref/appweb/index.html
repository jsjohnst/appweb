Programming APIs

The APIs are divided into two kinds: Ejscript APIs for developers to create Ejscript web applications and C APIs for system's developers to embed and extend Appweb itself.

Group Description
Appweb Native APIs C Language APIs
Ejscript Library Ejscript Language Library

Stability Classifications

Rather than providing a blanket backwards compatibility guarantee, APIs are classified according to their stability. These Stability Classifications should assist developers in migrating to new versions of Appweb.
